# My Gallery App


## Features in the application
- Fetch colors from Random Image
- Fetch labels from Random Image
- Fetch images from cache memory
- Edit image
- Delete Image
- Share Image
- Add Image from files
- Redirect Url option
- Create Recycle view
- Swip and Drag&Drop option
- Sort data 
- Search item
